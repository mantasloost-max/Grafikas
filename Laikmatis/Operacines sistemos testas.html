Laikmatis + resursų logas

Šitas HTML įrankis padeda kaupti matavimus: (1) laiką (Start→Stop) kiekvienai komandai, (2) resursus (CPU/RAM/SWAP) kiekvienai komandai atskirai ir (3) laiką per kiek pasiekta tavo stebima RAM apkrova (pvz. „10 langų“ scenarijus). Windows resursų jis neskaito automatiškai — tu suvedi skaičius iš Task Manager / Resource Monitor.
